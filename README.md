# CharesAPI

![Image of Helios]()

## Services

- Get product review by product_id

- Get review metadata by product_id

- Add new review

- Update helpfulness of a review

- Report a review

## Get started

The easiest way to get started is to pull the docker image. Go to INSERT_URL and click pull image.

```
$ npm install
```

```
$ npm start
```

```
Start Server
```
